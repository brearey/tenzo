# Arduino Projects - QWEN Context

## Обзор репозитория

Репозиторий содержит несколько Arduino-проектов, разработанных в образовательных и экспериментальных целях. Основные направления:

1. **tenzo** — система мониторинга целостности тросов (тензодатчик + HX711 + RGB-лента)
2. **wrestling-robot** — робот с 6 сервоприводами, управляемый по Bluetooth
3. **wrestling-test** — тестовая программа для демонстрации движений робота

## Технологии и оборудование

- **Платформа:** Arduino Uno, Arduino Nano
- **Датчики:** HX711 + тензодатчик мостовой 20 кг
- **Актуаторы:** Сервоприводы Tower Pro SG90, Feetech FS90
- **Связь:** Bluetooth модуль JDY-31 (аналог HC-06)
- **Индикация:** RGB-лента (5V, общий анод)
- **Питание:** LiPo аккумулятор 45C 1800 mAh, 7.4V
- **Библиотеки:** HX711, VarSpeedServo, SoftwareSerial

## Структура проектов

### tenzo (`tenzo/tenzo.ino`)

**Назначение:** Интеллектуальный детектор обрыва для натяжных конструкций (курсовой проект).

**Функциональность:**
- Калибровка базовой линии (10 секунд)
- Экспоненциальное сглаживание (EMA): `filtered = 0.88 * filtered + 0.12 * raw`
- Адаптивная компенсация дрейфа нуля (`BASELINE_ADAPT_ALPHA`)
- Детекция двух состояний:
  - **FULL_BREAK** — полный обрыв (падение ≥ 8.5%)
  - **PARTIAL_BREAK** — частичный разрыв (падение 3–7.5%)
- Подтверждение срабатывания по времени (500ms для полного, 300ms для частичного)
- RGB-индикация: синий (калибровка), зелёный (норма), жёлтый мигающий (частичный), красный мигающий (полный)
- Вывод данных для Serial Plotter

**Схема подключения:**

| Тензодатчик | HX711 |
|-------------|-------|
| Красный     | E+    |
| Черный      | E-    |
| Зеленый     | A-    |
| Белый       | A+    |

| HX711 | Arduino |
|-------|---------|
| GND   | GND     |
| DT    | A0      |
| SCK   | A1      |
| VCC   | 5V      |

| Arduino | RGB-лента |
|---------|-----------|
| 2       | R         |
| 3       | G         |
| 4       | B         |
| 13      | VCC       |

### wrestling-robot (`wrestling-robot/wrestling-robot.ino`)

**Назначение:** Управление роботом с 6 сервоприводами через Bluetooth (Android-планшет).

**Функциональность:**
- Приём команд формата `ID:Угол` через Bluetooth и USB
- Плавное движение серво через VarSpeedServo
- Индивидуальные диапазоны для Tower Pro (500–2500 мкс) и Feetech (544–2400 мкс)
- Асинхронное управление (не блокирует код)

**Схема подключения:**

| Arduino | Серво |
|---------|-------|
| 4       | Servo 1 (Tower) |
| 5       | Servo 2 (Tower) |
| 6       | Servo 3 (Feetech) |
| 7       | Servo 4 (Feetech) |
| 8       | Servo 5 (Feetech) |
| 9       | Servo 6 (Feetech) |

| Arduino | Bluetooth (JDY-31) |
|---------|--------------------|
| 3 (RX)  | TX                 |
| 2 (TX)  | RX                 |

**Питание:** Внешний LiPo 7.4V → VIN + серво напрямую. GND общий.

### wrestling-test (`wrestling-test/wrestling-test.ino`)

**Назначение:** Демонстрационная программа движений робота (приветствие, движения руками/ногами).

**Функциональность:**
- Последовательное выполнение хореографических элементов
- Разные скорости для ног (`LEG_SPEED=30`) и рук (`HAND_SPEED=80`)
- Автоматический запуск при старте

## Важные заметки

### Питание сервоприводов
- 6 серво потребляют до **1300 мА** при одновременной работе
- Напряжение: 7.54–7.64V
- **Критично:** при питании от Arduino через мотор-шилд при 700–800 мА происходит перезагрузка
- **Решение:** внешний LiPo аккумулятор, конденсатор 25В 470 мкФ параллельно питанию серво

### Библиотеки
- `HX711` — работа с тензодатчиком
- `VarSpeedServo` — плавное управление серво (скорость 1–255)
- `SoftwareSerial` — Bluetooth на пинах 2,3

### Разработка
- Код на русском языке (комментарии и вывод в Serial)
- Используется `F()` макрос для экономии RAM
- Форматирование: Arduino IDE style (2 пробела)

## Команды

- **Сборка:** Откройте `.ino` файл в Arduino IDE → Verify/Upload
- **Монитор порта:** Tools → Serial Monitor (9600 baud)
- **Serial Plotter:** Tools → Serial Plotter (для tenzo)
