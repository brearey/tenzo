# Список вопросов для формы регистрации на сайте «РобоХапсагай»

**Цель формы:** Сбор контактных данных от потенциальных клиентов (образовательных учреждений) для отправки коммерческого предложения и предоставления доступа к учебным материалам на платформе `coreapp.ai`.

---

### Поля формы:

1.  **Наименование образовательной организации***
    *   **Тип поля:** Текстовое поле (Text Input)
    *   **Пример:** `МБОУ "СОШ №3 г. Якутска"`
    *   **Описание:** Помогает идентифицировать учреждение, которое заинтересовано в продукте.

2.  **Ваше Имя и Фамилия***
    *   **Тип поля:** Текстовое поле (Text Input)
    *   **Пример:** `Иванов Иван Иванович`
    *   **Описание:** Имя контактного лица для дальнейшего общения.

3.  **Ваша должность***
    *   **Тип поля:** Текстовое поле (Text Input)
    *   **Пример:** `Учитель информатики`, `Директор`, `Руководитель "Точки Роста"`
    *   **Описание:** Позволяет понять, с кем ведется диалог и каков уровень принятия решений.

4.  **Рабочий E-mail***
    *   **Тип поля:** Поле для ввода Email (Email Input)
    *   **Пример:** `ivanov.i@school3.edu`
    *   **Описание:** **Ключевое поле.** На эту почту будет отправлено коммерческое предложение и предоставлен доступ к платформе `coreapp.ai`.

5.  **Контактный телефон***
    *   **Тип поля:** Поле для ввода телефона (Phone Input)
    *   **Пример:** `+7 (914) 123-45-67`
    *   **Описание:** Для оперативной связи и уточнения деталей заказа.

6.  **Какой пакет вас интересует?**
    *   **Тип поля:** Выпадающий список (Dropdown) или радио-кнопки (Radio Buttons)
    *   **Варианты:**
        *   Базовый (только материалы)
        *   Стандарт (с консультациями)
        *   Расширенный (с обучением педагогов)
        *   Комплексный (внедрение "под ключ")
        *   Требуется индивидуальный расчет
    *   **Описание:** Помогает предварительно понять потребности клиента.

7.  **Комментарий или вопрос**
    *   **Тип поля:** Большое текстовое поле (Textarea)
    *   **Пример:** `Планируем запустить кружок на 3 группы по 10 человек. Сколько комплектов нам понадобится?`
    *   **Описание:** Необязательное поле для дополнительных вопросов от клиента.

---
*Обязательные для заполнения поля отмечены звездочкой (*).*
